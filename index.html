About me

I'm a research mathematician with a background in programming and computer science. Over the past year, I've become passionate about the impacts that deep learning can have on society.

Github projects

Deep autoencoders and restricted Boltzmann machines

In this project I am reproducing results from the paper "Reducing the Dimensionality of Data with Neural Networks" by Hinton and Salakhutdinov. My code implements a restricted Boltzmann machine, pretrains a deep autoencoder using RBM's, then loads the weights into a Keras model for fine tuning.

Generative adversarial networks

A generative adversarial network for generating hand-drawn digits, trained on MNIST using Docker and AWS EC2.

Estimating Hofer-Zehnder capacities

A constrained gradient-descent algorithm (with non-linear constraints, using projected gradients) written in python with NumPy and SciPy.

Mathematics research

My mathematics research studies an intersection between dynamical systems, topology, and differential geometry (at a basic level, the tools I rely on boil down to analysis and linear algebra). Recently, I have been studying connections between Gelfand-Zeitlin systems and the exciting field of tropical geometry. You can read more about it on my University of Geneva department webpage.
